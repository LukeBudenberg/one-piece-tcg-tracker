Instructions to Generate Icons:

  1. Right-click on the icon preview above
  2. Select "Inspect" or "Inspect Element"
  3. Right-click on the <div class="icon-preview"> element
  4. Select "Capture node screenshot" (Chrome) or similar
  5. This saves a 512x512 image

Or use an online tool:

Visit PWABuilder.com and use their icon generator:

  1. Go to pwabuilder.com
  2. Click "Image Generator"
  3. Upload any image (can use a One Piece character or logo)
  4. Download the generated icon pack
  5. Extract to the icons/ folder
